M-Audio Axiom 25

This script was written for a Axiom 25 on a modified version of factory preset 1, i.e. the state of the keyboard after a factory reset.

Create the preset

In order to use this script all encoders must be set to "2â€™s Compliment from 64 / Relative (Binary Offset)" mode. this is accomplished by using the following process for each encoder.

  1. Press the Advanced button.
  2. Press Ctrl Assign key.
  3. Turn the encoder you wish to map.
  4. Enter 146 using the keys.
  5. Press the Enter key.
  6. Press the Data 2 key.
  7. Enter the CC number you would like the encoder to transmit.
  8. Press the Enter key.

After all the encoders have been mapped save the preset.

  1. Press the Advanced button.
  2. Press the Store key.
  3. Pick the location you would like to store your preset.
  4. Press the Enter key.

Keys

The keys are mapped as a note input named "Axiom 25 - Keys".

Drum Pads

The drum pads are mapped as a note input named "Axiom 25 - Pads".

Rotary encoders

The rotary encoders are by mapped to the macros of the primary device of the current track.

Transport buttons

  • Loop: Toggles the VST window if your primary device is a VST.
  • Rewind: Selects the previous track.
  • Forward: Selects the next track.
  • Stop: Stops playback and sets the play position to the start of the arrangement.
  • Play: Starts playback of the arrangement.
  • Record: Starts recording for armed tracks.
